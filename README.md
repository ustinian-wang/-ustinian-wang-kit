# `@ustinian-wang/kit`

## Introduction
- 鉴于多个项目都有相似的工具函数，为了逻辑复用，提供此工具包
- 如果lodash-es能满足需求，就不用在这里添加方法了

## Installation

```shell
yarn add @ustinian-wang/kit 
```

## Usage

```javascript
import { cloneDeep } from "@ustinian-wang/kit";

let oldObj = {
    a: 66
};
console.log( cloneDeep( oldObj ) );
// TODO: DEMONSTRATE API
```
